Evolución de la web

Alumno : Eduardo Varela Hernández

    Web 1.0

  • Solo se podía consumir contenido
  • Información unidireccional, es decir solo se podía acceder a ella sin la posibilidad de interactuar

    Web 2.0

  • Foros
  • Blogs
  • Comentarios
  • Redes Sociales
  • Permitía la posibilidad de compartir información

    Web 3.0

  • Web semántica
  • Uso del lenguaje en la red
  • Búsqueda de contenidos utilizando palabras clave

    Web 4.0

  • Comportamiento inteligente y predictivo
  • Se basa en poner en marcha un conjunto de acciones que tendrán como resultando aquello que pedimos, deseamos o decimos
